# Escalation Matrix

## Level 1 — Automated
- Duplicate charge under $100
- Password reset requests
- Subscription status inquiries
- Invoice requests

## Level 2 — Senior Agent
- Refunds over $100
- Billing disputes spanning multiple months
- Account access issues after failed identity verification
- Service downtime compensation requests

## Level 3 — Manager
- Legal or regulatory complaints
- Data deletion requests (GDPR/CCPA)
- Enterprise contract modifications
- Repeated escalations from same customer (3+ in 30 days)
